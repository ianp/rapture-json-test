rapture-json-test
=================

Contains a selection of tests for Rapture JSON, which test various extraction and pattern
matches on JSON. These tests currently run against the following JSON parsers:

 - Scala Standard Library JSON parser
 - Jawn
 - Jackson
 - Argonaut
 - JSON4S
 - Lift JSON
 - Spray JSON
